# 서식 지정자

자바 System.out.printf()

→ **출력 값을 서식에 맞춰서 출력**

### 서식 지정자

| 서식 지정자 | 출력 데이터 형태 |
| --- | --- |
| %c | 하나의 문자 |
| %s | 문자열 |
| %d | 부호 있는 10진 정수 |
| %i | 부호 있는 10진 정수 |
| %f | 고정 소수점으로 표현한 실수 |
| %o | 부호 없는 8진 정수 |
| %u | 부호 없는 10진 정수 |
| %x | 부호 없는 16진 정수(소문자) |
| %X | 부호 없는 16진 정수(대문자) |
| %e | 부동 소수점으로 표현한 실수(e-표기법) |
| %E | 부동 소수점으로 표현한 실수(E-표기법) |
| %g | 값에 따라 %f나 %e를 사용함 |
| %G | 값에 따라 %f나 %e를 사용함 |
| %% | 퍼센트 기호 출력 |
